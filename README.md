## Установка
### Windows
Скачайте релиз и запустите installer.exe, если
у версии нет installer.exe то скачивайте .exe файл
с названием версии
### Linux 
скачивайте tar.xz, deb или rpm пакет, если такого нет
то скачивайте файл с .x86_64 с название версии
